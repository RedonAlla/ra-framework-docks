Avatars can be used to represent people or objects. It supports images, Icons, or letters.

The **RA Avatar** is part of the `RA Framework` and is available under the `ra-avatar` NPM package.


## Dependencies
- `ra-theme-context`
- `ra-icons`
- `expo-image`
