**AvatarGroup** is part of the [**ra-avatar**](../) package.
To use the **AvatarGroup** you needs to install `ra-avatar` package. [Fallow this link](../#installation) on how to install the package.

## API

```jsx
import { AvatarGroup } from 'ra-avatar';
```

## Component

**AvatarGroup**

Type: `React.PureComponent<AvatarGroupProps>`

## Props

AvatarGroup extends [`ViewProps`](https://reactnative.dev/docs/view#props) from **React Native**.

|  Name   | Type | Required | Default Value | Description |
| -- | -- | -- | -- | -- |
| sortIndex | `string` | **false** | `asc` | `asc` or `desc` are supported |
| itemPadding | `number` | **false** | 13 |  |
| itemBorderWidth | [`BorderWidth`](docs/components/avatar/single/#borderwidth) | **false** | `thick` | Optional avatar borders width. |
| itemBorderColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | ``theme.colors.card`` | Borders color according `react-native` ColorValue. |
| children | Array of React.ReactElement of type [`AvatarProps`](/docs/components/avatar/single/#props) | **true** | `full` | |

## Playground

```jsx live noInline
//Play with props to see live changes;

const styles = {
  groupStyle: {
    paddingTop: 8,
    paddingBottom: 8,
    paddingRight: 8,
    paddingLeft: 21,
    borderRadius: 99,
    backgroundColor: '#ed143d30'
  }
};

render(
  <AvatarGroup style={styles.groupStyle}>
    <Avatar text='RA' size='large' type='text' color='primary' />
    <Avatar icon='avatar' size='large' type='icon' color='success' />
    <Avatar source="/img/logo.svg" size='large' color='info' />
  </AvatarGroup>
);
```
