:::note
**Spinner** is part of the [**ra-icons**](../icons) package.
To use the **RA Spinner** you needs to install `ra-icons` package. [Fallow this link](/docs/components/icons/ra-icons-basic-usage) on how to install the package.
:::

You can use any of paskage's icon as apinner.

## API

```jsx
import { Spinner } from '@ra/icons';
```

## Component

**Spinner**

Type: `React.MemoExoticComponent<(props: IconProps) => React.JSX.Element>`

## Properties

:::info INFO
`Icon` and `Spinner` components expects props of type `IconProp` which extends `TextProps` from `react-native`
:::

|  Name   | Type | Required | Default Value | Description |
| -- | -- | -- | -- | -- |
| name | [`IconName`](#iconname) | **true** | `undefined` |  |
| size  | [`IconSize`](#iconsize) | **false** | `'normal'` | Icon size |
| color  | [`IconColor`](#iconcolor) | **false** | `theme.colors.text` | Icon color one of color used by ThemeProvider or given ColorValue |
| TextProps  | [`TextProps`](https://reactnative.dev/docs/text#props) | **false** | `undefined` | **TextProps** from `react-native` |

### IconSize

* 'small' = 16
* 'normal' = 16
* 'medium' = 32
* 'large' = 48
* number

### IconColor

* 'default'
* 'primary'
* 'secondary'
* 'info'
* 'success'
* 'warning'
* 'error'
* 'dark'
* 'light'
* [ColorValue](https://reactnative.dev/docs/colors)
