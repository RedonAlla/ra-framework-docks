**Text Box** is part of the [**ra-inputs**](./) package.
To use the **Text Box** you needs to install `ra-inputs` package. [Fallow this link](./#installation) on how to install the package.

## API

```jsx
import { TextBox } from 'ra-inputs';
```

## Component

**TextBox**

Type: `React.PureComponent<TextBoxProps>`

## Props

import Highlight from '@site/src/components/Highlight';

`TextBoxProps` props extends [`TextInputProps`](https://reactnative.dev/docs/textinput#props) from `react-native`
so any [`TextInputProps`](https://reactnative.dev/docs/textinput#props) propertie from **React Native** it is applicable also in **Text Box** of `ra-buttons` package.

### InputType
- `outlined`,
- `underlined`

### Sizes
- `small`
- `medium`
- `large`
- `default`

### BorderRadius
- `none` => 0
- `small` => 2
- `medium` => 4
- `large` => 6
- `full` => 100%
- number => any number

### BorderWidth
- `none` => 0
- `hairline` => ([StyleSheet.hairlineWidth](https://reactnative.dev/docs/stylesheet#hairlinewidth))
- `thin` => 1
- `base` => 2
- `thick` => 3
- `number` => any number

### InputColor
* `default`
* `primary`
* `secondary`
* `info`
* `success`
* `warning`
* `error`
* `dark`
* `light`
* [`ColorValue`](https://reactnative.dev/docs/colors)
