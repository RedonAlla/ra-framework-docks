The CheckBox is typically used to represent boolean values via a binary checked state

**Check Box** is package is part of **ra-inputs** for React Native, it provides a same base inputs UI library and functionality.

**RA Check Box** is part of the [**ra-inputs**](./) package.
To use the **RA Check Box** you needs to install `ra-inputs` package.
[Fallow this link](./#installation) on how to install the package.

## Dependencies
- `ra-theme-context`
- `ra-icons`
- `ui-constants`

## Playground

```jsx live noInline
//Play with props to see live changes;

function CheckPlayground() {
  const [value, setValue] = useState(true);

  return (
    <Check value={value}
          label='Try check box'
          type='inverse'
          size='large'
          radius='large'
          color='primary'
          borderWidth='thick'
          onValueChange={() => setValue(!value)}
    />
  );
}

render(<CheckPlayground />);
```
