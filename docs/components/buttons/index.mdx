The Buttons Package is part of **RA Framework** for React Native, it provides a clickable UI library and functionality.

They include a variety of button types and styles that have extensive styles options. This flexibility allows you to quickly and easily create the exact button you need to fit your specific requirements for functionality and appearance.


## Dependencies
- `ra-theme-context`
- `ra-icons`
- `ui-constants`

import Installation from '../_installation.mdx';


```jsx

```

### BorderRadius
- `none` => 0
- `small` => 2
- `medium` => 4
- `large` => 6
- `full` => 100%
- number => any number

### BorderWidth
- `none` => 0
- `hairline` => ([StyleSheet.hairlineWidth](https://reactnative.dev/docs/stylesheet#hairlinewidth))
- `thin` => 1
- `base` => 2
- `thick` => 3
- `number` => any number

### ITextProps
```ts
export interface BaseTextProps extends TextProps {
  color?: ColorValue;
  activeColor?: ColorValue;
}

export interface ITextProps extends BaseTextProps {
  value: string;
};
```

### IconProps
```ts
export interface BaseTextProps extends TextProps {
  color?: ColorValue;
  activeColor?: ColorValue;
}

export type IconProps = {
  name: keyof IconName;
} & BaseTextProps;
```
