`ui-constants` Packages is part of **RA Framework**, that provides core constants related with UI Components.

## Installation

To use the **`ui-constants`** you need to download packages in your local storage, and then you must install the packages with npm locally.

For more details on how to install packages locally [fallow this link](/docs/getting-gtarted/installation).


Installing packages using npm:
```bash npm2yarn
npm install C:/ReactNative/framework/ui-constants
```
